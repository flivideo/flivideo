Provide a quick status summary of this FliVideo project.

## Instructions

Read the following files and provide a concise summary:

1. **Read** `docs/backlog.md` (requirements table)
2. **Read** `docs/changelog.md` (Quick Summary section)
3. **Read** `docs/brainstorming-notes.md` (Active Brainstorms)
4. **Read** `docs/handover-queue.md` (Pending/In Progress items)

**Note:** If any files don't exist, note that and skip that section.

```

### Recently Completed

### Backlog Highlights
```
